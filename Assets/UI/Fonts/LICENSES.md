# Demo font licenses

These fonts are used only by the demo scenes/pages under `Assets/` — they are
**not** part of the shipped `com.wevaui` package (which bundles its own OFL
fonts under `Packages/com.wevaui/Runtime/Resources/Fonts/`, each with an
adjacent `*-LICENSE.txt`).

All fonts here are licensed under the **SIL Open Font License 1.1**
(<https://scripts.sil.org/OFL>), which permits bundling and redistribution:

| Font | Source | License |
|------|--------|---------|
| Patrick Hand (`PatrickHand-Regular.ttf`) | <https://fonts.google.com/specimen/Patrick+Hand> | SIL OFL 1.1 |
| Sniglet (`Sniglet-ExtraBold.ttf`) | <https://fonts.google.com/specimen/Sniglet> | SIL OFL 1.1 |

`Assets/TextMesh Pro/Fonts/LiberationSans.ttf` ships with Unity's TextMesh Pro
package essentials (SIL OFL 1.1) and is included for the TMP text path.

If you fork this repo for redistribution, consider dropping each font's full
`OFL.txt` (from the source links above) next to its `.ttf`.
