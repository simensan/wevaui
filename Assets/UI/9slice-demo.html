9-Slice Demo

Three ways to use 9-slice images in Weva. The same source asset (a sprite with borders configured in the Unity Sprite Editor) drives all three.

Asset setup: import a frame PNG, set Sprite Mode = Single, open the Sprite Editor, drag the green border handles to mark the slice insets, save. Register it in your InMemoryImageRegistry with registry.Register("UI/PanelFrame", new SpriteImageSource(sprite)). The engine reads sprite.border automatically.

Where to get free 9-sliceable frames: Kenney's UI Pack (kenney.nl/assets/ui-pack — CC0), OpenGameArt 9-patch frames.

Method A — <img> with native 9-slice

Just an <img>. The engine detects the sprite's border and paints 9 parts. No CSS needed.

Verification

All three methods produce 9 FillRect paint commands (4 corners, 4 edges, 1 center) when the source has nine-slice metadata. Open the DevTools overlay to inspect the paint command list, or run ImgPaintTests.Img_with_native_9slice_source_emits_9_parts for the unit-test version.
